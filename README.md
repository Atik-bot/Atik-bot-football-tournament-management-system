# Atik-bot-football-tournament-management-system
# ⚽ Football Tournament Management System

A database management system project developed for the **Introduction to Database** course at  
American International University-Bangladesh (AIUB).

This project manages football tournaments, teams, players, matches, leagues, stadiums, and awards using SQL and database normalization concepts.

---

 Features

- Team Management
- Player Information Management
- Match Scheduling
- Tournament Management
- League Management
- Stadium Records
- Award Tracking
- SQL Queries
- Joins and Subqueries
- Database Views
- Normalization (UNF → 3NF)

---

## Technologies Used

- Oracle SQL
- SQL Developer
- ER Modeling
- Database Normalization

---
